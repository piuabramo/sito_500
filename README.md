# sito_500

Questo repository contiene una semplice pagina statica per l'associazione "Miez Mil" di Gonars.

Per visualizzare il sito localmente apri il file `index.html` nel browser oppure usa un server statico (consigliato):

```powershell
# da PowerShell nella cartella del progetto
python -m http.server 8000; Start-Process http://localhost:8000
```

I file principali creati:
- `index.html` — pagina principale
- `css/styles.css` — stili base

Modifiche future suggerite: aggiungere immagini in `assets/`, aggiornare i codici della lotteria con i dati reali, e collegare i contatti ufficiali.
